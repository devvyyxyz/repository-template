## Description

This repository serves as a template for creating structured markdown files for GitHub repositories.

## File Structure

```bash
project-root
├── README.md
├── LICENSE
├── CODE_OF_CONDUCT.md
├── .github
│   ├── FUNDING.yml
│   ├── PULL_REQUEST_TEMPLATE.md
│   └── ISSUE_TEMPLATE
│       ├── bug_report.md
│       └── feature_request.md
├── .gitignore
```
